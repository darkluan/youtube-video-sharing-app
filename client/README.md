# Starting Project

### Create config file

Create `.env` file. Copy content from `.evn.example` into `.env`. Change config in `.env` corresponding to your environment

### Install package

```
npm i
```

### Run

```
npm run dev
```

### build

```
npm run build
```

- Default app will be started at http://localhost:3000

### Testing

```
npm run cypress:o
```
